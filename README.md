# SendTree Bank

SendTree Bank 是一個基於 ASP.NET MVC 的網上銀行平台，提供貸款申請、帳戶管理、交易記錄等功能。本專案旨在提供一個易於使用的界面，幫助用戶輕鬆管理其金融業務。

## 後臺功能

- **貸款商品管理**
- **貸款申請審核**
- **貸款還款計畫管理**。
- **交易記錄查詢**。

## 技術棧

- **前端**：
  - HTML5
  - CSS3
  - JavaScript
  - jQuery
  - Bootstrap

- **後端**：
  - ASP.NET MVC
  - C#
  - Entity Framework
  - SQL Server

- **第三方庫**：
  - SweetAlert
  - Chart.js
